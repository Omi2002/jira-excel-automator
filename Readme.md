# 🚀 Jira Excel Automator

A powerful Python tool to bulk create and update Jira tasks directly from Excel/CSV files.

## ✨ Features
- **Sync Logic:** Automatically detects if a task exists (Updates) or needs to be created (New).
- **Assignee Management:** Maps users via Account IDs for 100% accuracy.
- **Original Estimate:** Supports Jira's `timeoriginalestimate` field.
- **Web UI:** Built with Streamlit for a no-code user experience.

## 🛠️ Installation
1. Clone this repo: `git clone https://github.com/your-username/jira-excel-automator.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
